== README

This is a simple rails application that demonstrates the use of DropzoneJs  and paperclip to upload 
images using ajax in a rails app.

Dependencies you will have to fullfill include

* ImageMagick (For paperclip gem to work)

For the complete tutorial visit http://josephndungu.com/tutorials/ajax-file-upload-with-dropezonejs-and-paperclip-rails

Check out the demo at http://stark-castle-1300.herokuapp.com/
